hexo-algolia的environment variable配置

发表于 2017-11-06 | 更新于: 2017-11-06 | 分类于 hexo-algolia |

使用hexo搭建博客的确十分方便快捷，而且网上教程很多，按照教程一步步来即可轻松完成博客搭建。

在搭建博客的过程中还是碰到了一些小问题，其中有一个就是hexo-algolia环境变量配置问题。

按照网上教程，现在algolia官网注册好账号，新建index，得到 ID ，API key等，配置好根目录文件 _config.yml里的algolia项，在git bash中执行hexo algolia,报错:

查阅hexo-algolia官方文档后发现，这是因为新版hexo-algolia(版本^1.2.3)需要用户提供一个单独的(separate) API key来设置environment variable，具体步骤如下:
