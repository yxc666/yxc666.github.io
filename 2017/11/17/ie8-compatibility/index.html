ie8兼容性问题防坑指南

发表于 2017-11-17 | 更新于: 2017-11-17 | 分类于 ie浏览器 ， 兼容性问题 |

近期项目客户要求兼容ie8,于是我们开始填坑之路,在此记录我们遇到的问题和解决办法,以作总结。

CSS3 Media Query

ie8无法识别Media Query,所以可以hack一下,引入Respond.js解决此问题。

nth-child

first-child是css2的内容,但nth-child和last-child不是,所以ie8不兼容,我们可以给元素设置不同的class,还可以使用+来代替nth-child。

ie8&7&6下使用css3属性

使用PIE.htc！首先,介绍一下PIE.htc文件,它是用于控制浏览器行为,.htc文件目前只有ie浏览器支持,它可以让ie支持以下属性:

  • border-radius
  • box-shadow
  • Gradients
  • RGBA
  • background-size
